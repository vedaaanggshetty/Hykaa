### Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.

### Run Locally

To run **Hykaa** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/vedaaanggshetty/Hykaa.git
```

Windows:

```bash
git clone https://github.com/vedaaanggshetty/Hykaa.git
```
